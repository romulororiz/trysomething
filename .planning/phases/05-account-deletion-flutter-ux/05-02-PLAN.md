      final String url;

      } else {
        url = 'https://play.google.com/store/account/subscriptions';
      }

    **7. Add `_executeDeleteAccount` method** on `_SettingsScreenState` -- the core orchestration (per D-07, D-08, D-09).

    ```dart
    Future<void> _executeDeleteAccount(BuildContext context, WidgetRef ref, {String? password}) async {
      final success = await ref.read(authProvider.notifier).deleteAccount(password: password);

      if (!context.mounted) return;

      if (success) {
        // Clear SharedPreferences (per D-08 -- superset of logout)
        final prefs = ref.read(sharedPreferencesProvider);
        await prefs.clear();

        // Reset onboarding state (per D-08)
        ref.read(onboardingCompleteProvider.notifier).reset();

        // Show snackbar BEFORE navigating (per D-07; avoids context issues)
        showAppSnackbar(context, message: 'Account scheduled for deletion', type: AppSnackbarType.info);

        // Navigate to login (per D-07)
        if (context.mounted) context.go('/login');
      } else {
        // Show error, stay on Settings (per D-09)
        showAppSnackbar(context, message: 'Failed to delete account. Please try again.', type: AppSnackbarType.error);
      }
    }
    ```

    IMPORTANT: The `deleteAccount()` provider method (from Plan 01) already clears tokens, RevenueCat, analytics, Hive, and sets auth to unauthenticated. This method adds the remaining cleanup that requires WidgetRef: SharedPreferences.clear() and onboarding reset.
