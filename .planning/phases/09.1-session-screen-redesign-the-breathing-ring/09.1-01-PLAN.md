**A. Create `lib/components/breathing_ring.dart`:**

Create a `BreathingRingPainter` CustomPainter with these parameters:
- `double progress` (0.0 to 1.0)
- `double breathScale` (default 1.0, range ~1.0 to 1.008)
- `double glowIntensity` (default 0.15, 0.25 for last minute per D-24)
- `double ringOpacity` (default 1.0, 0.4 when paused per D-21)

Paint method draws 4 layers in order:
1. **Track ring (D-05):** `canvas.drawCircle(center, radius, trackPaint)` with `color: Color(0x0FFFFFFF)` (6% white), `PaintingStyle.stroke`, `strokeWidth: 4.5`
2. **Glow halo (D-01):** Only when `progress > 0`. `canvas.drawArc(rect, startAngle, sweepAngle, false, glowPaint)` with `color: AppColors.accent.withValues(alpha: glowIntensity * ringOpacity)`, `strokeWidth: 8.0`, `strokeCap: StrokeCap.round`, `maskFilter: MaskFilter.blur(BlurStyle.normal, 8.0)`
3. **Progress arc (D-01, D-02):** Only when `progress > 0`. `canvas.drawArc(rect, startAngle, sweepAngle, false, arcPaint)` with `color: AppColors.accent.withValues(alpha: ringOpacity)`, `strokeWidth: 4.5`, `strokeCap: StrokeCap.round`
4. **Leading dot (D-04):** Only when `progress > 0 && progress < 1.0`. Calculate dot position at arc tip using `cos`/`sin` of `startAngle + sweepAngle`. Draw soft glow behind: `drawCircle(dotPos, 8, Paint()..color=AppColors.accent.withValues(alpha: 0.15 * ringOpacity)..maskFilter=MaskFilter.blur(BlurStyle.normal, 4))`. Draw solid dot: `drawCircle(dotPos, 3, Paint()..color=AppColors.accent.withValues(alpha: 0.8 * ringOpacity))`

All angles start from `-pi / 2` (12 o'clock). `sweepAngle = 2 * pi * progress`.

Apply breathing scale via `canvas.save()`, `canvas.translate(center.dx, center.dy)`, `canvas.scale(breathScale, breathScale)`, `canvas.translate(-center.dx, -center.dy)`, then `canvas.restore()` after drawing.

`shouldRepaint` compares all 4 fields.

Create a `BreathingRing` StatefulWidget wrapping the painter:
- Constructor params: `double progress`, `Duration breathCycleDuration` (default 4000ms per D-10), `double glowIntensity` (default 0.15), `double ringOpacity` (default 1.0), `double ringSize` (default 270 per D-03)
- State owns:
  - `_breathController`: AnimationController with `duration: breathCycleDuration` that `repeat()`s
  - `_pulseController`: AnimationController with 600ms duration for milestone pulse (D-23)
  - Listener on `_breathController` calculates `_breathScale = 1.0 + 0.008 * sin(_breathController.value * 2 * pi)` and calls `setState`
- `didUpdateWidget`: If `breathCycleDuration` changed, stop controller, set new duration, restart `repeat()` (per Pitfall 3 in research)
- Build: `RepaintBoundary` wrapping `SizedBox(width: ringSize, height: ringSize, child: CustomPaint(painter: BreathingRingPainter(...)))`
- Public method `triggerMilestonePulse()` that animates `_pulseController.forward(from: 0.0)` and adds pulse scale on top of breath scale: `finalScale = breathScale + pulseScale` where pulseScale rises to 0.012 at halfway through _pulseController and back to 0

Import `dart:math` for `sin`, `cos`, `pi`. Import `app_colors.dart`.
