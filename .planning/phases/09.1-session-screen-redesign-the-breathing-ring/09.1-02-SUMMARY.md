# Phase 09.1 Plan 02: Session Screen 5-Layer Stack & Ring-Centric Timer Summary

**One-liner:** 5-layer cinematic Stack with breathing background, ambient gradient, film grain, and persistent BreathingRing across all phases; timer phase rewritten with Phosphor glass controls, coach tip bottom sheet, and elegant completion celebration.

## What Was Done

### Task 1: Rewrite session_screen.dart with 5-layer cinematic Stack

Replaced the 2-layer Stack (Container placeholder + phase content) with a 5-layer cinematic Stack:

1. **Layer 1 - Breathing background (D-09):** Custom `_BreathingBackground` widget using `TweenAnimationBuilder<double>` with `onEnd` callback to toggle direction, continuously pulsing between `#0A0A0F` and `#0F0F14` synced with the ring's breath cycle duration.

2. **Layer 2 - Ambient gradient spotlight (D-07):** `RadialGradient` centered at `Alignment(0, -0.16)` (~42% from top), coral at 4% opacity fading to transparent.

3. **Layer 3 - Film grain overlay (D-08):** `FilmGrainOverlay` from Plan 01 at 1.5% opacity.

4. **Layer 4 - Breathing ring (D-01 through D-05):** `BreathingRing` from Plan 01, positioned at `height * 0.42 - 135` (centering the 270dp ring at ~42% from top). Wrapped in `AnimatedOpacity` for paused dimming to 40% (D-21).

5. **Layer 5 - Phase content:** Same `AnimatedSwitcher` with fade+slide transition as before.

Added helper methods:
- `_breathCycleDuration()` - Returns 4000ms active (D-10), 8000ms paused (D-11), 3000ms last minute (D-12)
- `_glowIntensity()` - Returns 0.25 for last minute (D-24), 0.15 otherwise
- Milestone pulse trigger at 50% progress via `_ringKey.currentState?.triggerMilestonePulse()` (D-23)

**Commit:** `f9c6a6c`

### Task 2: Rewrite session_timer_phase.dart with ring-centric layout

Rewrote the timer phase to compose around the ring (which now lives in session_screen.dart):

- **Overline (D-17):** `"HOBBY . step title"` format — hobby uppercase, step in original case, middle dot separator, 2-line max with ellipsis
- **Rolling timer (D-22):** Preserved `_RollingTimer`, `_RollingDigit`, `_Separator` with font size bumped from 64pt to 72pt IBM Plex Mono, digit dimensions from 40x72 to 44x80
- **Warm shift (D-24):** Timer digits shift to `Color(0xFFF5E6D8)` when remaining < 60s
- **Completion celebration (D-13):** PhosphorIcons.checkCircle (bold, success color), "Session complete" in display style, optional completionMessage — all staggered fade-in
- **Paused state (D-21):** "Paused" label fades in, "End session early" link fades in with IgnorePointer when not paused
- **Step instructions (D-18):** Below ring area, AppTypography.body, centered, constrained to 72px max height
- **Coach tip (D-19):** 36dp glass circle with PhosphorIconsLight.lightbulb, only visible when `session.coachTip != null`, opens `showAppSheet` with tip text
- **Glass pause/play button (D-20):** 56dp glass circle (`glassBackground` + `glassBorder`), PhosphorIconsBold.pause/play icons (22px, textPrimary), AnimatedSwitcher for icon swap, AnimatedScale 0.97 on press

Deleted: `_PausePlayButton`, `_PauseIconPainter`, `_PlayIconPainter` (replaced by Phosphor icons in glass circle).
