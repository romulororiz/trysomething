# Phase 9.1: Session Screen Redesign — The Breathing Ring - Context

**Gathered:** 2026-03-22
**Status:** Ready for planning

<domain>
## Phase Boundary

Complete visual redesign of the 4-phase session screen (Prepare → Timer → Reflect → Complete). Replace the amateur 103-particle field with a premium progress ring that carries across all phases. The session should feel cinematic, immersive, and premium — like Headspace meets Apple Watch.

</domain>

<decisions>
## Implementation Decisions

### Ring visual style
- **D-01:** Confident weight — Apple Watch style. 4-5px progress arc stroke, visible glow halo (8px at 15% opacity), clear progress at a glance
- **D-02:** Always coral (`AppColors.accent` / `#FF6B6B`) — consistent brand accent, NOT category-colored
- **D-03:** Ring diameter ~260-280dp, centered vertically (slightly above true center, ~42% from top)
- **D-04:** Leading dot at the arc tip — 6dp filled circle at 80% opacity with soft glow
- **D-05:** Track ring (full circle background) at 6% white opacity

### Background treatment (replaces particles)
- **D-06:** Kill all 103 particles — delete `particle_timer_painter.dart` entirely
- **D-07:** Ambient gradient spotlight — soft radial gradient centered on ring, coral at 3-5% opacity
- **D-08:** Film grain texture overlay — subtle static noise at 1-2% opacity for cinematic warmth
- **D-09:** Breathing background — entire background subtly pulses between `#0A0A0F` and `#0F0F14`, synced with ring breathing animation

### Ring breathing animation
- **D-10:** Active: 4-second sinusoidal cycle, scale 1.0 to 1.008
- **D-11:** Paused: 8-second cycle (slower, calmer)
- **D-12:** Last minute: 3-second cycle (gentle urgency)

### Completion celebration
- **D-13:** Claude's discretion — make it premium, elegant, animated, smooth, and seamless. Check if hobbies have `completionMessage` in the database/schema and use it.
- **D-14:** 3-second auto-transition to Reflect (keep current timing)
- **D-15:** NO journey progress bar on completion — keep it clean

### Phase transitions
- **D-16:** Claude's discretion — pick the most cinematic approach. The ring should be the visual constant that carries across phases (preview in Prepare, fills in Timer, blooms in Complete). Other content crossfades around it.

### Session context during timer
- **D-17:** Show hobby name + step name as overline above the ring (e.g. "POTTERY · Wedging your first clay ball")
- **D-18:** Show step instructions below the ring — brief how-to text in `AppTypography.body`
- **D-19:** Coach tip accessible on-demand — small subtle icon (e.g. lightbulb or chat bubble) that opens a bottom sheet overlay with the coach tip for the current step. Does NOT pollute the main timer screen.

### Controls
- **D-20:** Pause/resume: 56dp glass circle button with Phosphor icon (`pause` / `play`), standard 0.97 scale-on-press
- **D-21:** When paused: "Paused" label appears above button, "End session" text link fades in below. Ring dims to 40% opacity.
- **D-22:** Timer digits: IBM Plex Mono 72pt inside the ring, keep existing `_RollingTimer` component (the rolling digit animation is good)

### Milestone moments
- **D-23:** Halfway (50%): gentle ring pulse (scale 1.02, 600ms) + light haptic
- **D-24:** Last minute: timer digits warm-shift to `Color(0xFFF5E6D8)`, ring glow intensifies to 25%
- **D-25:** Completion: ring bloom + haptic pattern (keep existing 4-hit pattern)

### Claude's Discretion
- Exact completion celebration animation (bloom, glow, check icon — make it premium)
- Phase transition animation approach (ring morph recommended)
- Film grain implementation (shader vs image overlay vs custom painter)
- Whether to keep `category_shape_painter.dart` or delete with particles
- Exact coach tip icon placement and bottom sheet design
- Prepare phase layout around the static ring preview

</decisions>

<specifics>
## Specific Ideas

- The session should feel like Headspace meets Apple Watch — warm, focused, unmistakably premium
- The ring is the ONLY visual anchor — everything else orbits around it
- Empty space around the ring is powerful — don't fill it with noise
- The breathing animation should be almost imperceptible but prevent the screen from feeling dead
- Film grain adds the "cinematic" quality that matches the design system name "Warm Cinematic Minimalism"
- The coach tip should be discoverable but not visible by default — a subtle icon that doesn't compete with the ring

</specifics>
