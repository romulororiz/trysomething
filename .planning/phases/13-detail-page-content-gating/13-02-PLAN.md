**Create `lib/components/pro_gate_section.dart`:**

A `StatelessWidget` (not ConsumerWidget -- it receives `isLocked` as a param, no need to read providers itself) that wraps any child with a blur + lock overlay when locked.

Constructor parameters:
- `required Widget child` -- the actual content widget
- `required bool isLocked` -- whether to show blur overlay
- `required String sectionTitle` -- displayed on overlay (e.g., "Why people stop")
- `required String teaserText` -- one-line teaser (e.g., "Know the common traps before you start")
- `required BuildContext parentContext` -- for calling showProUpgrade (or accept an `onTap` callback instead; simpler: accept `VoidCallback? onLockTap`)

Implementation when `isLocked == false`: return `child` directly (zero overhead).

Implementation when `isLocked == true`:
1. Wrap in `GestureDetector` with `onTap` calling the provided `onLockTap` callback
2. Use `ClipRRect` with `borderRadius: BorderRadius.circular(20)` to contain blur
3. Inside Stack:
   - First child: `ImageFiltered(imageFilter: ImageFilter.blur(sigmaX: 8, sigmaY: 8), child: child)` -- blurs the real content (NOT BackdropFilter -- see RESEARCH.md Pitfall 1 about scroll jank)
   - Second child: `Positioned.fill` with a semi-transparent overlay (`AppColors.background.withValues(alpha: 0.55)`) containing centered Column:
     - Lock icon: `Icons.lock_outline_rounded`, size 24, `Colors.white.withValues(alpha: 0.5)` (subtle per CONTEXT.md specifics)
     - SizedBox(height: 8)
     - Section title: `AppTypography.sansLabel` with `AppColors.textPrimary`
     - SizedBox(height: 4)
     - Teaser text: `AppTypography.sansTiny` with `AppColors.textSecondary`, `textAlign: TextAlign.center`
     - SizedBox(height: 12)
     - Coral "Unlock with Pro" pill: `Container` with `AppColors.accent` background, `BorderRadius.circular(20)`, horizontal padding 16, vertical 8, white text using `AppTypography.caption.copyWith(color: Colors.white, fontWeight: FontWeight.w600)`

Import `dart:ui` for `ImageFilter`, import theme files, import glass_card if needed.
