<tasks>

<task type="human-action">
  <name>Task 1: Extract release SHA fingerprints from keystore</name>
  <files>none</files>
  <action>
Run this command to extract the SHA-1 and SHA-256 fingerprints:

```bash
"/c/Program Files/Android/Android Studio/jbr/bin/keytool.exe" -list -v \
  -keystore android/keystores/trysomething/upload-keystore.jks \
  -alias upload \
  -storepass "TrySomething2026!"
```

Copy the SHA-1 and SHA-256 fingerprint values (they look like `AA:BB:CC:DD:...`).

Then in **Firebase Console** (https://console.firebase.google.com):
1. Go to Project Settings → Your apps → Android app
2. Click "Add fingerprint"
3. Add BOTH the SHA-1 and SHA-256 from the output above
4. Save

After adding fingerprints, click **"Download google-services.json"** and replace the file at `android/app/google-services.json`.

**Tell Claude** when done, providing the SHA-1 fingerprint (needed for Google OAuth).
  </action>
  <verify>
    <automated>test -f android/app/google-services.json && echo "PASS: google-services.json exists" || echo "FAIL"</automated>
  </verify>
  <done>Firebase has release SHA-1 + SHA-256, google-services.json re-downloaded</done>
</task>

  <action>
Collect these 3 keys:

**PostHog API Key:**
1. Go to https://us.posthog.com (or your PostHog instance)
2. Project Settings → API Key
3. Copy the project API key (starts with `phc_...`)

**Sentry DSN:**
1. Go to https://sentry.io
2. Project Settings → Client Keys (DSN)
3. Copy the DSN URL (looks like `https://xxx@xxx.ingest.sentry.io/xxx`)

**Google OAuth Server Client ID:**
1. Go to Firebase Console → Authentication → Sign-in method → Google
2. The "Web client ID" is your GOOGLE_SERVER_CLIENT_ID
3. Copy it (looks like `xxx.apps.googleusercontent.com`)

**Tell Claude** all 3 values.
